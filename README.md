## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- modern browser (e.g. Chrome, Firefox)
- text editor (e.g. VSCode)
- CLI/terminal

### Setup

```
git clone git@github.com:virag-ky/React-Tutorial---To-Do-App.git
```

### Install

```
npm install
```

### Open in browser

```
npm run start
```
